# entzündungverstehen.de

Eine Bildungswebsite die erklärt wie chronische Entzündung im Körper funktioniert — und was man selbst tun kann.

## Ziel

Nicht "warum ich?" — sondern: was passiert in meinem Körper, und was kann *ich* tun?

Viele Menschen mit chronischen Entzündungserkrankungen bekommen von ihrem Arzt Medikamente — aber keine Erklärung. Diese Website füllt diese Lücke.

## Inhalt

- **Start** — Grundprinzip: gesunde vs. chronische Entzündung, animierte Kaskade
- **😴 Schlaf** — Mechanismus + konkrete Maßnahmen
- **🥗 Ernährung** — Darm/Mikrobiom + Blutzucker als zwei Entzündungswege
- **💪 Bewegung** — Myokine, Tregs, Übertraining-Warnung
- **🧘 Stress** — HPA-Achse, MBSR, NSDR-Video
- **🔬 Erkrankungen** — MCAS, Rheumatoide Arthritis, Colitis Ulcerosa

## Prinzipien

- Informieren, nicht diagnostizieren
- Kein Schuld-Framing ("warum ich?")
- Medizinisch korrekt, aber verständlich
- Lebensstil ergänzt Medikamente — ersetzt sie nicht
- Dark Mode unterstützt
